You will submit you work through GitHub. Your submission should contain a report in PDF format, a README explaining which files to run, and your Python's files.

The README file should be called README.md and should be written in Markdown.

To write our report you can use Word, Markdown, or LaTeX. For Markdown I recommend Typora, for LaTeX you can either install it on your computer or use Overleaf.
